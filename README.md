

# 📸 Responsive Photo Gallery

✨ *2k24 was the best year of my life, filled with memories I never wanted to lose. As a BCA 2nd year student, I used my web designing skills to build this photo gallery — a creative digital space to hold and relive those special moments.*

---

## ✨ Features

* 📱 **Responsive Design** → Works on desktop, tablet, and mobile
* 🎨 **Modern Grid Layout** → Auto-fit images with dynamic spans
* 🖼️ **Hover Effects** → Smooth zoom-in with brightness animation
* 🌄 **Background Styling** → Full-page image background with fixed effect
* ⚡ **Pure HTML + CSS** → Lightweight and beginner-friendly

---

## 📷 How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/responsive-photo-gallery.git
   ```
2. Open the folder and add **your own photos** inside it.

   > The gallery was originally built with around **60 photos**.
   > You can use the same format (`p1.jpg`, `p2.jpg`, …) or update the file paths in `index.html`.
3. Open `index.html` in your browser to view your personal gallery.
